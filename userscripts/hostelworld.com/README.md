# Hostelworld Enhancer

This userscript adds a review search panel to Hostelworld property pages so review text is easier to search and compare.

## What it does

- loads hostel reviews into an on-page search panel
- supports live regex filtering with highlighted matches
- adds quick search links to common external searches
